    Visual Explanation of Python Panda Library

    If you’ve gone through the visual explanation of Numpy Library, there is one thing common in Numpy and panda that they both are used to store data and perform operations on that data.

    For a quick difference, remember Numpy as a group of lists or multi-dimensional arrays whereas remember Panda as a table. We call those tables, Data Frame.

    Pandas offer 2 data structures;

    1. Series (one-dimensional)
    2. DataFrame (two-dimensional)

    Each column in a data frame is a series. You can consider a data-frame as a database table where we can query data just like SQL.

    • NumPy: Use NumPy when you need to perform numerical computations, especially with arrays or matrices. It’s ideal for mathematical operations, linear algebra, and scientific computing.
    • Pandas: Use Pandas when working with structured or tabular data. If you need to manipulate, clean, analyze, or visualize data stored in tables or CSV files, Pandas is the go-to library.

    Things to learn You can majorly divide your learning in 4 parts

    1. Build Data frame: either in code itself or from some external file like Excel, CSV etc. Adding or deleting rows or columns.
    2. Selection of data: Select a particular row or column, or a range. Group by data etc.
    3. Operation on data: Like aggregation, Handling NA values etc.
    4. Save Dataframe to a file: for future use.
